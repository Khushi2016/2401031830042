# 🚀 Aura Academy – AI Coding Tutor

## 📌 Project Description

Aura Academy is an AI-powered coding tutor platform designed to provide an interactive and modern learning experience. It helps users improve programming skills through structured modules, quizzes, and collaborative features.

---

## ✨ Features

* 🎯 Interactive Quiz System (Arena)
* 💬 Real-time Chat UI (Co-op Study)
* 📊 Dashboard with progress tracking
* 📚 Programming Languages Module
* 👤 Profile with achievements & certificates
* ⭐ Reviews & Feedback system
* 📝 Blogs & learning resources

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, Bootstrap
* **Scripting:** JavaScript, jQuery
* **Design:** Figma
* **Version Control:** GitHub
